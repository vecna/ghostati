How this archive is maintained.

REFERENCES.json is the source of truth for this page. Add, edit, or remove an entry there, then run:

node scripts-dev/validate-references.js && node scripts-dev/build-references-page.js

The build script regenerates the reference list below from templates/references.template.html — this header, the intro copy, and this section are hand-edited directly in that template file and are never overwritten.

Technical tests are only one part of the work. If you know of facial recognition being used in public space, use the reporting page to share what you know, only if it is safe for you to do so.
